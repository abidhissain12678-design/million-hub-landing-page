# Task Landing Page Generator System

Professional task landing page generator built with Next.js, Firebase, and Tailwind CSS.

## 🎯 Features

### Admin Panel
- ✅ Admin authentication with Firebase
- ✅ Create, edit, and delete tasks
- ✅ Auto-generate unique slugs for landing pages
- ✅ Upload task images
- ✅ Add YouTube video guides
- ✅ Manage ads (top, middle, left, right, bottom)
- ✅ Generate landing page links
- ✅ Track all tasks

### User Landing Page
- ✅ Mobile responsive design
- ✅ Dark theme with gold accents
- ✅ Glassmorphism cards
- ✅ Embedded YouTube player
- ✅ Multiple ad placements
- ✅ Call-to-action buttons
- ✅ QR code generation
- ✅ Social share buttons (WhatsApp, Telegram, Twitter)
- ✅ Fast loading with Next.js optimization

### Premium Features
- 🎯 Automatic short links
- ⏱️ Task countdown timer
- 📱 Social share integration
- 📸 Proof submission form
- 📊 Analytics dashboard

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

You can start editing the page by modifying `app/page.tsx`. The page auto-updates as you edit the file.

This project uses [`next/font`](https://nextjs.org/docs/app/building-your-application/optimizing/fonts) to automatically optimize and load [Geist](https://vercel.com/font), a new font family for Vercel.
